## 📖 Descripción

Aplicación web interactiva donde el usuario puede jugar al clásico juego de **Piedra, Papel o Tijera** contra la máquina.

Permite introducir un nombre de usuario, seleccionar número de partidas y visualizar el resultado de cada ronda.



## 🚀 Características principales

- Validación de nombre y número de partidas  
- Selección visual de jugada  
- Generación aleatoria de jugada de la máquina  
- Historial de resultados en tiempo real  
- Botón de reinicio de partida  



## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript

## ⚙️ Funcionamiento

El juego funciona mediante eventos en JavaScript:

- El usuario introduce sus datos  
- Selecciona una jugada  
- La máquina genera una opción aleatoria  
- Se calcula el resultado y se guarda en el historial
